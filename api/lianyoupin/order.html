# 兑换区商品兑换

方式

POST

路径

/api/v1/orders

参数

名称 类型 必须 说明
sku_id int 是 商品 sku_id
pay_pass string 是 支付密码
consignee_id string 是 收货人 id
num int 是 数量

响应

Status code 200

{
  "msg": "购买成功",
  "code": 0 /*注意，如code为：1000-未设置支付密码 1001-支付密码错误 客户端需要另行处理交互*/,
  "data": {
    "order_id": 1 /*订单号*/
  }
}
